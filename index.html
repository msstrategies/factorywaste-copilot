Narration settings

By default the Copilot uses a deterministic mock narrator that speaks only from the computed facts (no key, fully offline). You can optionally route narration through the Claude API.

Whether mock or LLM, the narrator can only state numbers the deterministic analysis layer computed. The LLM is given the grounded facts and is instructed to narrate them and invent nothing. The key is never written to disk and never leaves your browser except in the direct call to Anthropic.

Grounding eval suite

10 question / expected-fact pairs. Each asserts on the structured facts the narrator is allowed to speak from, so a green run means the Copilot cannot state a wrong number.
